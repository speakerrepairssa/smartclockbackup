ℹ️ How it works

1. Connect via relay – The browser calls a VPS relay server which forwards the request to your Hikvision device using Digest authentication. This works with local LAN IP addresses.

2. Extract month by month – Events are fetched one month at a time for the selected range.

3. Map – Each event is matched to an employee by badge number or name. Unmatched events are skipped and shown below.

4. Save – Matched events are written directly to Firebase under businesses / {businessId} / attendance_events / {date} / {slot} with duplicate prevention.

Note: Employee names on the device must roughly match the names in SmartClock for mapping to succeed. The import is safe to run multiple times – existing events are not duplicated.
